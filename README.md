# Student Performance Analysis
Overview

This project analyses a student perfomane dataset to understand the influence of various factors on student marks

Tools Used:
Python
Pandas
Matplotlib

Data Preparation:
Before the analysis, the dataset was explored and checked for:
Missing values
Duplicate records
Data types and column information
Summary statistics

Analysis performed:
1. Data exploration and cleaning
2. Lunch type vs student performance
3. Gender vs student performance
4. Test preparation vs student performance
5. Average score by subject
6. Parental education vs student performance
7. Reading score vs writing score correlation

Key findings:
Students who completed the test preparation course are found to have scored higher on average.
Students whose parents had higher education levels tended to score higher
Students with a standard lunch generally had highr scores
Reading had the highest average score among the three subjects
Reading and writing scores were strongly related
